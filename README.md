# PassOP - Your Personal Password Manager 🔐

PassOP is a modern web application designed to help you store and manage your website credentials locally. It provides a simple interface to save, edit, delete, and copy your passwords with ease.

## 🚀 Live Demo
[View the Live Project Here](https://astoriaking15.github.io/PassOP-Manager/)

## ✨ Key Features
- **LocalStorage Storage:** Your data never leaves your browser; it's saved locally for privacy.
- **Copy to Clipboard:** One-click copy for usernames and passwords.
- **Password Visibility:** Toggle password masking for better security while viewing.


## 🛠️ Built With
- React.js (Vite)
- Tailwind CSS
- Lucide React (Icons)
- React-Toastify (Notifications)

## 📖 Installation
1. Clone the repo: `https://github.com/astoriaking15/PassOP-Manager.git`
2. Install dependencies: `npm install`
3. Start the dev server: `npm run dev`
